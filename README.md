## 📌 Project Overview

This project demonstrates a complete **Exploratory Data Analysis (EDA)** and **statistical hypothesis testing** workflow using Python.

The analysis focuses on a wine dataset containing chemical features such as:

* Alcohol
* Malic Acid
* Ash
* Wine Cultivar

The goal of the project was to explore the dataset, understand feature behavior, detect outliers, visualize distributions, and perform statistical tests to compare different wine cultivars.

---

## 🎯 Project Goals

* Understand the structure and quality of the dataset.
* Perform initial data inspection using NumPy and Pandas.
* Identify missing values and possible outliers.
* Analyze numerical and categorical features.
* Visualize data distributions.
* Apply statistical hypothesis testing.
* Draw conclusions based on statistical evidence.

## 🔍 Workflow

### 1. Data Import and Inspection

The dataset was loaded and inspected using both **NumPy arrays** and **Pandas DataFrames** to understand the structure, feature ranges, and sample distribution.

### 2. Data Cleaning

The dataset was checked for missing values and categorical inconsistencies. No missing values were found.

### 3. Exploratory Data Analysis

Descriptive statistics were calculated for the main numerical features in order to understand:

* Value ranges
* Mean and standard deviation
* Feature variability
* Possible abnormal values

### 4. Outlier Detection

Box plots were used to identify potential outliers. A small number of extreme values were removed carefully in order to reduce noise without damaging the dataset structure.

### 5. Data Visualization

Visualizations were created to better understand the distributions of the features and the differences between wine cultivars.

### 6. Statistical Hypothesis Testing

Statistical tests were performed to compare chemical properties between wine cultivar groups and evaluate whether observed differences were statistically significant.

---

## ❓ Research Questions

### Question 1

Is the mean **malic acid level** equal between wine cultivar 1 and wine cultivar 2 at a 5% significance level?

### Question 2

Is there a statistically significant difference between the mean **ash level** of wine cultivar 1 and wine cultivar 2 at a 2% significance level?

---

## 📊 Key Results

* No missing values were found in the dataset.
* Outliers were detected using box plots.
* The mean malic acid level between cultivar 1 and cultivar 2 was not found to be significantly different at the 5% significance level.
* The mean ash level showed a statistically significant difference between cultivar 1 and cultivar 2 at the 2% significance level.
